# dialtrader
NMT Dial Trader
DialTrader is a single-file HTML/JS watch trading app hosted at app.dialtrader.com, built on Supabase. The stack is vanilla JS + Supabase JS client, no frameworks. Current version is v15. Features include: inventory management, 10-stage pipeline, CRM, invoices, calendar (FullCalendar), tasks, tax report, storefront. Landing page is at dialtrader.com.
